# CustomizedVariants and Crystal

This file tracks the most interesting open area in the wrapper research.

## Main question

Can `CustomizedVariants` / `Crystal` carry useful custom visuals?

## Current best answer

Probably yes for visuals, but not yet proven as a complete replacement for archetype gameplay semantics.

More specifically:

- some wrappers look mostly link-only
- some evidence suggests a `Crystal` can carry real geometry

That makes `Crystal` interesting for:

- hiding or replacing ugly visible parts of water carriers

But it does not yet prove:

- that `Crystal` alone can create fully custom gameplay water

## Practical meaning

The most promising use is still:

- let a known-good archetype provide gameplay water
- let custom wrapper-side geometry improve the visuals
