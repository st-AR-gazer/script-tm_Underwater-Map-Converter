# Crystal Editing Hypotheses

This file records the main hypotheses around wrapper-side geometry editing.

## Best current hypothesis

If `Crystal` can carry useful geometry, then the most promising use is:

- keep a known-good water archetype for gameplay water
- use wrapper-side geometry to improve or replace the visible mesh

## Why this matters

That would be much more practical than needing `Crystal` to carry the entire gameplay-water system by itself.

## Current status

- interesting
- plausible
- not yet fully proven
