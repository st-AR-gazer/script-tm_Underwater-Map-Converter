# GreenCoast Water

GreenCoast is one of the most useful environments for the practical underwater-map goal.

Why?

Because its working water-carrying block set appears much broader than BlueBay or WhiteShore.

## The basic system

GreenCoast centers water on a lake-zone system:

- `Lake.EDFlat`
- `LakeClip.EDClip`
- `Lake.ZoneFlat`

## Local water base

Key local carrier:

- `GreenCoast\Media\Prefab\Zone\Lake\Base.Prefab.Gbx`

Key materials:

- `GreenCoast\Media\Material\Water.Material.Gbx`
- `GreenCoast\Media\Material\Lake.Material.Gbx`

## Deco lake blocks

GreenCoast also has local classic lake blocks such as:

- `DecoLakeCurve2In`
- `DecoLakeCurve2Out`
- `DecoLakeCurve3In`
- `DecoLakeCurve3Out`

These matter because they prove:

- not all water in the environment is just the global lake zone
- some prefabs carry local water-rendered geometry directly

## Practical working pattern

User testing indicated a broad set of working families, including:

- `DecoLake*`
- `Open*RoadStraightOnLakeShore*`
- `Platform*BaseOnLakeShore*`
- `Road*OnLakeShore*`
- `Road*OnLakeShoreStraight*`

## Practical takeaway

If your goal is "give me a broad working palette for underwater-map building", GreenCoast is one of the best current candidates.
